## RU

### Натуральный продукт 0% ИИ-содержащий, без вайбкод-добавок, ГМО-кода и фреймворковых подсластителей
Минималистичный консольный вычислитель, созданный для практики и закрепления знаний. Только питон, терминал, желание получить опыт и знания.

**Возможности:**
- **Элементарные операции**: сложение, вычитание, умножение, деление, возведение в степень, корни.
- **Финансовые**: расчёт процентов, платежей по кредиту, переплаты.
- **Статистика**: среднее, медиана, мода, дисперсия, стандартное отклонение.
- **Тригонометрия**: градусы/радианы, sin/cos/tan, аркфункции.
- **Теория чисел**: проверка на простоту, факториал, числа Фибоначчи, НОД/НОК.
- **Системы счисления**: конвертация между DEC, BIN, OCT, HEX.

<a name="english"></a>
## EN

### 100% Organic, 0% AI-generated. No vibe-code additives, GMO-logic, or framework sweeteners
A minimalist console calculator built for pure practice and skill-honing. Just Python, a terminal, and a raw desire to actually learn how to code instead of letting a chatbot do the work.

**Features:**
- **Elementary operations**: addition, subtraction, multiplication, division, exponentiation, roots.
- **Finance**: percentage calculations, loan payments, overpayments.
- **Statistics**: mean, median, mode, variance, standard deviation.
- **Trigonometry**: degrees/radians, sin/cos/tan, arc functions.
- **Number theory**: prime checking, factorial, Fibonacci, GCD/LCM.
- **Numeral systems**: conversion between DEC, BIN, OCT, HEX.
